# README #

Svelte Component Library

## Install Dependencies
```shell
pnpm install
```

## Add Dependencies
```shell
pnpm install <pkg> -w
```

## Add Dependencies for scoped packages
```shell
pnpm install <pkg> -r --filter <pkg_selector>
```

## Publish
```shell
lerna publish
```

## Usage

### Launch component website
```shell
npm run dev
```

### run storybook
```shell
cmex sb
```

### build components
```shell
cmex build
cmex build --scope <component>
```

### run unit test
```shell
cmex test
cmex test --scope <component>
```
