# README #

Svelte Component Library

## Install Dependencies
```shell
pnpm install
```

## Add Dependencies
```shell
pnpm install <pkg> -w
```

## Add Dependencies for scoped packages
```shell
pnpm install <pkg> -r --filter <pkg_selector>
```

## Publish
```shell
lerna publish
```

## Usage
```shell
npm run dev
cmex sb
cmex dev
cmex test
```
