# Button


| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|------|
| mode | 按钮主题 | day/night | day |
| type | 按钮类型 | primary/default/reverse/text | primary |
| size | 按钮大小 | big/medium/small | medium |
| loading | 载入状态 | boolean | false |
| disabled | 失效状态 | boolean | false |
| on:click | 点击按钮回调 | (event) => void | - |
