# Button

## Embedding svelte inside a mdx file

The following content renders the svelte component - hello.svelte below.

import toReact from "svelte-adapter/react";
import Button from "@cmexd/button/src/index";

export const baseStyle = {
  width: "100%"
};

export const WrapButton = toReact(Button, baseStyle, "div");


<WrapButton label="test" primary size="large" onClick={()=> {console.log(123)}}/>
