# Intro

Let's discover **CMEXD in less than 5 minutes**.

## Getting Started

Get started by **creating a new site**.

## Generate a new site

[Node.js](https://nodejs.org/en/download/) version 14 or above

Run the following command to install the CLI:

```bash
npm install -g @cmex/cli
```
